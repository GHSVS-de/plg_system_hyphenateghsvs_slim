Die CSS-Selektoren, die Sie in dieses Feld zeilenweise eingeben, erhalten eine CSS-Klasse "hyphenate". Der Inhalt des markierten HTML-Elements und seiner untergeordneten Elemente wird getrennt. Das Markup ist dasselbe wie in einer CSS-Datei. Sie können HTML-Tags oder CSS-Klassen (mit einem führenden Punkt) oder IDs (mit einem führenden #) oder fast jede andere übliche Kombination eingeben.

Beispiel Tag body

Der sichtbare Inhalt aller HTML-Elemente innerhalb der Seite wird getrennt, da der übergeordnete <body> -Tag eine Klasse "hyphenate" (<body class="hyphenate">) erhält und alle untergeordneten Elemente eines so markierten Elements das Verhalten erben. Normalerweise müssen Sie dann viele der untergeordneten Elemente mit der Einstellung "Stopp-Hyphenate-Selektoren" ausschließen, um die Silbentrennung für diese zu deaktivieren.

Es wird nicht empfohlen, "body" zu konfigurieren, wenn Sie Leistungsprobleme vermeiden möchten. Das bedeutet, dass Sie den body-Tag nur für erste Tests verwenden sollten.

Sie sollten kleinere Container definieren und spezifischer. In Joomla ist das normalerweise nur die Ausgabe von Komponenten wie einem Artikel.

Tipp für Faule:

  • Umgeben Sie <jdoc:include type="component" /> in Ihrer template-index.php mit einem zusätzlichen "div" -Container. 
    <div class="makeMeHyphenated">
<jdoc:include type="component" />
</div>
  • Dann gebe Sie in das Feld .makeMeHyphenated ein.
  • Sie können auch <div class="hyphenate">schreiben und in dieses Feld einen "Unsinns-Selektor" eingeben (siehe unten warum), aber geben Sie in das Feld niemals .hyphenate ein!
Beispiel mit CSS-Klasse .item-page

Alle Tags (Elemente) mit Klasse item-page bekommen automatisch eine zusätzliche Klasse hyphenate. <div class="item-page something-else"> wird zu <div class="item-page something-else hyphenate">

Beispiel mit CSS-id #icons

Ein Element <div id="icons" class="small some-other-class"> wird automatisch zu <div id="icons" class="small some-other-class hyphenate">

Geben Sie 1 Selektor pro Zeile ein.

Einige Beispiele:

.item-page
.item p
#icons.small
div.container
section.somewhere div.inside
#myid .outside
[itemprop="headline"]
[itemprop="articleBody"]

Wenn Sie dieses Feld leer lassen und das Feld "Stopp-Hyphenate-Selektoren" ebenfalls leer ist, lädt das Plugin NICHT die Silbentrennungsskripte. Geben Sie einen "Unsinns-Selektor" wie "nonsensexyzzz" ein, wenn Sie das Laden erzwingen möchten, z.B. wenn Sie es vorziehen, die Klassen "hyphenate" und "donthyphenate" manuell in Ihr HTML-Markup einzufügen.
