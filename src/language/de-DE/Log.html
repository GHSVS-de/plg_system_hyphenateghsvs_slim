Unabhängig von den oben erwähnten Hyphenopoly-Konsole-Meldungen, können Sie hier weitere Log-Meldungen aktivieren, die in eine Datei geschrieben werden, wenn Sie hier "JA" wählen.

Letztlich liefert dieses Feature nicht mehr Informationen als die Hyphenopoly-Konsole-Meldungen. Aber in Kombination mit Einstellung "Bereinigung" protokolliert es auch Seitenlinks mit, was vielleicht hilfreich ist.

Man muss nicht für jede Seite einzeln in die Browser-Fehler-Konsole schauen, sondern kann über mehrere Seiten hinweg Meldungen sammeln. Auch über mehrere Tage hinweg. Aktivieren Sie das Feature und klicken Sie im Frontend ein paar Seiten durch und Sprachen (falls Sie eine mehrsprachige Seite verwenden). Schauen Sei dann hier, was gesammelt wurde.

Wie die Datei heißt und wo sie abgelegt wird und seine Dateigröße, erfahren Sie durch Klick auf den Knopf "Show Log File Path and Size".

Beachten Sie, dass das Anzeigen sehr großer Dateien mit der Schaltfläche "Show Log File Content" sehr lange dauern kann. Seien Sie geduldig, nachdem Sie auf den Button geklickt haben!

Kopieren Sie den Dateiinhalt in einen Editor oder laden Sei die Datei per FTP herunter und löschen Sie sie bevor sie zu groß wird. Da sich das "Loggen" Feature bemüht jede Meldung nur ein mal zu protokollieren, können übergroße Dateien zu einer Verlangsamung des Systems führen.

Deaktivieren Sie "Loggen", wenn die Debug-Arbeit abgeschlossen ist.

Siehe auch Einstellung "Konsolenmeldungen deaktivieren".

Siehe auch Einstellung "Bereinigung".
