Die CSS-Selektoren, die Sie in dieses Feld eingeben, beenden die Silbentrennung. Sie bekommen die Klasse "donthyphenate". Der Inhalt des markierten HTML-Elements und seiner untergeordneten Elemente wird daher nicht getrennt. Das Markup ist dasselbe wie in einer CSS-Datei. Weitere Informationen finden Sie in der Beschreibung zur Einstellung "Hyphenate-Selektoren".

Beachten Sie: Sie können eine Klasse "donthyphenate" nicht durch eine Klasse "hyphenate" in einem untergeordneten Element aufheben, wenn die Silbentrennungs-Skripts die Silbentrennung steuern. Das bedeutet, dass das die Inhalte des Elements "P" in dem folgenden Codeausschnitt nicht getrennt werden, da das umgebende "div" die Klasse "donthyphenate" hat.

<div class="someDiv donthyphenate">
	<p class="someP hyphenate">
		Some text that will not be hypheanated!
	</p>
</div>

Wenn Sie dieses Feld leer lassen und das Feld "Hyphenate-Selektoren" ebenfalls leer ist, lädt das Plugin NICHT die Silbentrennung-Skripte. Siehe auch Beschreibung von "Hyphenate-Selektoren".
