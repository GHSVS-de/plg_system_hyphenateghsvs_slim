Das Plugin PLG_SYSTEM_HYPHENATEGHS hilft dabei, Silbentrennung in Ihrer Webseite zu integrieren. Gerade auf schmalen Bildschirmen wird Text unleserlich, wenn lange Wörter über den Seitenrand ragen oder abgeschnitten werden.

Andererseits gibt es Wörter, die eine Zeile halb füllen und das nächste längere Wort rutscht in die nächste Zeile, anstatt die Zeile ähnlich einem Blocksatz aufzufüllen und dann getrennt zu werden.

Das Plugin HYPHENATEGHS hat zweierlei Grundfunktionen.

1.) Einsetzen von CSS-Klassen "hyphenate" und "donthyphenate" in den Quelltext einer Seite. Wo diese Klassen eingefügt werden, können Sie konfigurieren. Internet-Browser, die die CSS-Eigenschaft "hyphens" verstehen, also Silbentrennung ohne zusätzliche Tools beherrschen, benötigen nur noch ein paar Zeilen CSS, um die Inhalte der so konfigurierten Bereiche der Seite zu trennen oder nicht.

Viele Browser unterstützen die "hypens"-Eigenschaft nicht, andere unterstützen sie nicht für alle in der Site verwendeten Sprachen. Wenn das der Fall ist, kommt Funktion 2.) in's Spiel.

2.) Zuschalten von Silbentrennungs-Skripten (Hyphenation-Skripte) nur dann, wenn ein Browser die "hyphens"-Eigenschaft gar nicht unterstützt oder eine Seite eine Sprache verwendet, die der Browser nicht unterstützt.

Die von Ihrer Website verwendeten Sprachen müssen Sie in diesem Plugin konfigurieren.

Während Funktion 1.) auch alleine verwendet werden kann, benötigt Funktion 2.) immer auch Funktion 1.).
