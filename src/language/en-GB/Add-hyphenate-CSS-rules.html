This plugin adds always the CSS classes "hyphenate" and "donthypenate" to the elements that you've configured in fields "Hyphenate-Selectors" or "Do-Not-Hyphenate-Selectors".

Browsers (like FireFox) that are clever enough to hyphenate without additional scripts need the following CSS lines. You can add them in your CSS files yourself or just select YES here to load them automatically inside the <HEAD> of your template.

If you have no idea what I'm talking about then setting YES is recommended.
