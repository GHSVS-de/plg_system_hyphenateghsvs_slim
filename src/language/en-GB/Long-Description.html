Whenever I use the verb "hyphenate" that means "splitting a word with a hyphen at line end". I don't know a correct word for it.

The plugin PLG_SYSTEM_HYPHENATEGHS helps to integrate hyphenation in your website. Especially on narrow screens, text becomes unreadable when long words protrude over the page margin or are cut off.

On the other hand, there are words that half fill a line and the next longer word slips to the next line, instead of filling the line like a block set and then getting hyphenated.

The plugin HYPHENATEGHS has two basic functions.

1.) Insert CSS classes "hyphenate" and "donthyphenate" into the source code of a page. You can configure where these classes are inserted. Internet browsers that understand the CSS "hyphens" property, meaning hyphenation without additional tools, only need a few lines of CSS to hyphenate the contents of the configured sections of the page.

Many browsers do not support the "hypens" property, others do not support them for all used languages of your website. If that's the case, feature 2.) comes into play.

2.) Include hyphenation scripts only if a browser does not support the "hyphens" property or if a page uses a language that the browser does not support.

The languages used by your website must be configured in this plugin.

While function 1.) can also be used alone, function 2.) always requires function 1.).
