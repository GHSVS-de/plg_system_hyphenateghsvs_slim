The CSS selectors that you enter in this field will stop the hyphenatation. They get the class "donthyphenate". Thus the content of the marked HTML element and it's child elements will not be hyphenated. The markup is the same like in a CSS file. See the description of setting "Hyphenate-Selectors" for details.

Be aware: You can't cancel a class "donthyphenate" by a class "hyphenate" in a child element if the hyphenation scripts are controlling the hyphenation. That means that the element "P" will not be hyphenated in the following code snippet because the surrounding "div" has a class "donthyphenate".

<div class="someDiv donthyphenate">
	<p class="someP hyphenate">
		Some text that will not be hypheanated!
	</p>
</div>

If you leave this field empty and field "Hyphenate-Selectors" is empty, too, the plugin will NOT load the hyphenation scripts. See also description of "Hyphenate-Selectors".
